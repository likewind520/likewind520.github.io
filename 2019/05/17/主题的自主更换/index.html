主题的自主更换

发表于 2019-05-17 | 分类于 前端 | 评论数： | 阅读次数：

1.主题更换,以next为例

首先,去GitHub选择合适的主题hexo-theme-next,复制ssh下载地址.在blogs中gitbush输入git clone 地址.(http://theme-next.iissnan.com/getting-started.html#activate-next-theme) .next主题文档

其次,把下载的文件hexo-theme-next放到themes文件夹里面去.

最后,把文件的文件名放到_config.yml配置里面去.如:theme: hexo-theme-next

2.更改语言为中文

在_config.yml的language中改为zh-Hans后,发现并没有变化,去看了hexo-theme-next主题中languages的配置,发现只有zh-CN,把zh-Hans改为zh-CN就可以了.

3.增加标签页和分类页

在主题hexo-theme-next的根目录中的_config.yml里.找到menu.去掉tags和categories的”#”即可.

4.正确打开标签页和分类页

若此时打开标签页,会报错,出现一段路径.怎么正确打开呢?

输入hexo n page tags ,会在source下新建tags文件,然后在tags下的index.md中修改,如

title: 标签 date: 2019-05-17 10:29:08

type: “tags” (这里要写,不然不会出现暂无标签字样)

分类页同上,输入hexo n page categories

5.主题样式scheme的更改(内置了四种样式)

6.更改头像

首先把喜欢的图像放在主题根目录下的source的images中.

其次在主题的根目录中_config.yml里的avatar设置图片路径:url: /images/avatar.gif

7.更改标题和作者

在博客blogs更目录下的_config.yml里的Site

title: Love author: LiMing Hu

8.启用侧边栏社交链接

在主题的根目录下的_config.yml里找到social去掉”#”设置即可.

9.添加友情链接

在主题的根目录下的_config.yml里找到Links去掉”#”设置添加链接即可.

10.开启打赏功能

在主题的根目录下的_config.yml里找到reward_settings去掉”#”设置即可.如:

11.开启微信公众号的订阅功能

在主题的根目录下的_config.yml里找到wechat_subscriber去掉”#”设置即可

12.设置图像为圆形可旋转

在主题的根目录下的source下的css里的_common下,找到components里的sidebar,用编辑器代开sidebar-author.styl,(不要用记事本,容易乱码).

.site-author-image {
     border-radius: 50%;
       transition: 2s all;            (图片圆角,2秒转一圈)
}
.site-author-image:hover {
     transform: rotate(360deg);     (鼠标移上去的hover效果)
}

13.首页文章设置阅读全文

两种方法,先说简单的:只需要在你的编辑器中要被截断的文字后面输入<!-- more -->即可.剩下的部分只有点击了”阅读全文”才出现.(这是单篇文章的设置).

另外一种,在在主题的根目录下的_config.yml里找到auto_excerpt:

enable: false
length: 150 (这是多篇文章的统一设置).

14.给博客添加fork GitHub

首先去http://tholman.com/github-corners/上找一个喜欢的图标,复制右边的内容.

其次用编辑器打开主题根目录下的layout文件下的_layout.swig.

最后找到<div class="headband"></div>,在其后面粘贴.更改你的GitHub地址,保存.

执行hexo clean=>hexo g=>hexo s 开启服务器测试即可.

15.修改文章内链接样式

首先是markdown的语法写出:[连接变色](地址)如下:[链接变色](http://www.baidu.com)

其次编辑器打开source=>css=>_common=>components=>post=>post.styl

修改其内容:

> 这个符号表示引用的内容

.post-body p a{ (不考虑为什么这样写)
color: red;
border-bottom: none;
&:hover{
color: yellow;
}
}
链接变色

16.修改文章底部带#的标签,更改图标样式

首先找到要修改代码的地方?用编辑器打开主题根目录下的layout下_macro里的post.swig.

搜索tag找到如下内容:<div class="post-tags"><a>#</a></div>在#的这个地方写入icon调用代码:<i class="fa fa-google"></i>保存.

最后再执行hexo clean=>hexo g=>hexo s 开启服务器测试即可.

字体图标库中文网

17.增加valine评论功能

了解valine评论功能
Valine 诞生于2017年8月7日，是一款基于Leancloud的快速、简洁且高效的无后端评论系统。 理论上支持但不限于静态博客，目前已有Hexo、Jekyll、Typecho、Hugo、Ghost 等博客程序在使用Valine。
用valine之前先去leanClound注册,得到APPID和appkey

具体操作如下:

首先去leanClound的控制台创建一个应用.在创建好的应用右上角的齿轮设置其应用key.复制id和key
其次编辑器打开主题根目录下的_config.yml,找到valine,改为true,粘贴ID和key,保存.
最后再执行hexo clean=>hexo g=>hexo s 开启服务器测试即可.
评论时的头像及其他设置,可参考valine文档.目前我选用7种非自定义图像中的wavatar

18.增加搜索功能(next主题有四种:Swiftype/微搜索/Local Search/Algolia),以Local Search为例,其他三种相对麻烦些.

1.安装 hexo-generator-searchdb，在博客的根目录下执行以下命令：

$ npm install hexo-generator-searchdb --save

2.编辑博客根目录下_config.yml，新增以下内容到任意位置：

search:
     path: search.xml
     field: post
     format: html
     limit: 10000

3.编辑博客根目录下的_config.yml，启用本地搜索功能：

local_search:
  enable: true

4.开启服务即可.
