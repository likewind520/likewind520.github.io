主题的自主更换

发表于 2019-05-17 | 分类于 前端

1.主题更换,以next为例

首先,去GitHub选择合适的主题hexo-theme-next,复制ssh下载地址.在blogs中gitbush输入git clone 地址.(http://theme-next.iissnan.com/getting-started.html#activate-next-theme) .next主题文档

其次,把下载的文件hexo-theme-next放到themes文件夹里面去.

最后,把文件的文件名放到_config.yml配置里面去.如:theme: hexo-theme-next

2.更改语言为中文

在_config.yml的language中改为zh-Hans后,发现并没有变化,去看了hexo-theme-next主题中languages的配置,发现只有zh-CN,把zh-Hans改为zh-CN就可以了.

3.增加标签页和分类页

在主题hexo-theme-next的根目录中的_config.yml里.找到menu.去掉tags和categories的”#”即可.

4.正确打开标签页和分类页

若此时打开标签页,会报错,出现一段路径.怎么正确打开呢?

输入hexo n page tags ,会在source下新建tags文件,然后在tags下的index.md中修改,如

title: 标签 date: 2019-05-17 10:29:08

type: “tags” (这里要写,不然不会出现暂无标签字样)

分类页同上,输入hexo n page categories

5.主题样式scheme的更改(内置了四种样式)

6.更改头像

首先把喜欢的图像放在主题根目录下的source的images中.

其次在主题的根目录中_config.yml里的avatar设置图片路径:url: /images/avatar.gif

7.更改标题和作者

在博客blogs更目录下的_config.yml里的Site

title: Love author: LiMing Hu

8.启用侧边栏社交链接

在主题的根目录下的_config.yml里找到social去掉”#”设置即可.

9.添加友情链接

在主题的根目录下的_config.yml里找到Links去掉”#”设置添加链接即可.

10.开启打赏功能

在主题的根目录下的_config.yml里找到reward_settings去掉”#”设置即可.如:
